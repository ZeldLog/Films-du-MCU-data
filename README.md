# Films-du-MCU-data

## Idée

- Moyenne / Médian nombre du nombres d'avis sur un films noté sur IMDB
- Notation de la franchise comparée au autres (Harry Potter, Stars Wars,... )
- Evolutions Notation des films au fils du temps + marquages temporel important (mort de Stan Lee, Les différentes phase du MCU )
- Notation des films en fonctions des acteurs / personnages dedans
- Nombre de place vendu des films Marvel / ou un films particulier ( le plus populaire des FILMS Marvel) 
-  Notation Films vs Série
- Rentabilité (bénéfices obtenue - productions du films/série)


## Configuration

### Outils
- Conda
- Numpy
- Panda

### Environnement
#### Conda :
```bash
conda env create -f Environnement/MCU-environment.yml --name MCU_env
conda activate MCU_env 
```
